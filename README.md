# markdown-contents
Creates table of contents for a markdown document
# Example
## Input file 
```
#My Project   
## Idea  
content  
## Implementation  
### Step 1  
content  
### Step 2  
content
```

## Output
```1. [My Project](#my-project)  
	1. [Idea](#idea)  
	2. [Implementation](#implementation)  
		1. [Step 1](#step-1)  
		2. [Step 2](#step-2)  

# My Project  
## Idea  
content  
## Implementation  
### Step 1  
content  
### Step 2  
content```
